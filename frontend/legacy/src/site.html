Sobre Nós

LunarLog é um projeto que simula um sistema de rastreamento de encomendas, desenvolvido como atividade avaliativa da disciplina de Estrutura de Dados do 3º Semestre do curso de Ciência da Computação da UNIJORGE.

O projeto foi desenvolvido por:

  • Iago de Carvalho
  • Nicole Flávia
  • Pedro Alexandre Costa
  • Ruan Santos
  • Valesca Lemos

Através das plataformas de desenvolvimento:

  • Front-End: Svelte,tailwind css , html, Java Script
  • Back-End: Java e Spring Boot
  • Banco de Dados: PostgreSQL

LunarLog

A LunarLog é uma empresa comprometida com a qualidade e a responsabilidade em todos os seus setores de serviço. Priorizar a experiência do consumidor é a nossa principal missão.
